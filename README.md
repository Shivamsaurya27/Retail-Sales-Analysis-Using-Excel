📊 Retail Sales Analysis using Excel

Overview

This project analyzes retail sales data using Microsoft Excel to uncover business insights related to sales performance, profitability, customer behavior, regional performance, and discount impact.

The project follows a complete data analysis workflow including data cleaning, KPI analysis, exploratory data analysis (EDA), visualization, and dashboard creation.

Key KPIs

- Total Sales: $2.29M
- Total Profit: $598K
- Total Orders: 5,009
- Profit Margin: 26.1%
- Unique Customers: 793

Key Insights

- Office Supplies generated the highest profit margin (32.8%).
- Furniture generated strong sales but the lowest margin (18.9%).
- Discounts between 1%–40% significantly reduced profitability.
- West region led in revenue but had the lowest margin.
- Customer concentration risk was low due to a broad customer base.
- A major data quality issue was identified in the Order Date field.

Business Recommendations

- Review discount policies to improve margins.
- Investigate pricing strategy in the West region.
- Increase focus on high-margin Office Supplies products.
- Improve data quality controls and governance.
